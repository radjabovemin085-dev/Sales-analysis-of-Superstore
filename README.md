# Sales-analysis-of-Superstore
Power BI Project
Superstore Sales Analysis

The project covers sales anaylsis of a fictional Superstore. This project explores sales trends, segment and category performance, product contributions, and revenue metrics using pandas, matplotlib, seaborn, and squarify.

Headings

Total Sales by Year – Bar plot showing yearly sales trends.

Sales by Segment, Region, and Category – Pie charts highlighting contributions of different segments, regions, and categories.

Sales by Sub-Category – Tree map to visualize the share of each product sub-category in total sales.

Sales Trend Over Time – Line chart showing sales evolution over months/years.

Average Revenue per Order (ARO) – Bar plot ranking sub-categories by average revenue per order.

Libraries used

pandas – Data manipulation

numpy – Numerical operations

matplotlib – Plotting graphs

seaborn – Statistical data visualization

squarify – Tree map visualization
